Revolution Realty API

Welcome to the Revolution Realty Django Backend API. This is the modernized version of the original Laravel application.

Available Endpoints:

GET / - This homepage
GET /sell-my-home - Sell home page
POST /saveContact - Save contact form
GET /property-search - Property search
GET /agent-list - Agent listings
GET /vendor-list - Vendor listings
GET /blog - Blog posts
GET /contact-us - Contact form

Technology Stack:

  • Backend: Django 5.2.4 with Python 3.11
  • Frontend: React with Vite, Tailwind CSS, and shadcn/ui
  • Database: MySQL with migrated data from Laravel
  • Features: Modern responsive design, improved performance, clean code architecture
